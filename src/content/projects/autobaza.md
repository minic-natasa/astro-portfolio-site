## Overview

Autobaza is a regional automotive marketplace serving Serbia, Montenegro, and Hungary — positioned as a "new era of online vehicle commerce." The platform connects private sellers and buyers through a modern, feature-rich interface built for serious vehicle transactions.

## What I Built

**Vehicle Marketplace**

Developed the full marketplace platform with detailed vehicle listings — covering brand, model, year, mileage, fuel type, emission class, registration status, import country, damage assessment, and pricing. Listings are displayed in a clean grid layout optimized for quick scanning and comparison.

**Advanced Search & Filtering**

Implemented a comprehensive filtering system allowing buyers to narrow down by vehicle type, condition, engine specs, safety features, location, and price range — reducing time-to-find for buyers and improving listing visibility for sellers.

**User System**

Built multi-role user functionality including personal listings management, a favorites/watchlist, direct messaging between buyers and sellers, a notification system, and full profile management.

**Rewards System**

Integrated a points-based rewards system to incentivize engagement — users earn points through platform activity, encouraging repeat visits and listing quality.

## Tech Stack

- **CMS:** WordPress
- **Frontend:** HTML, CSS, JavaScript
- **Features:** Marketplace listings, advanced filtering, user messaging, rewards system
